Pipeline

How a piece graduates.

Blender poly base → sculpt pass for ornaments and hero props. Procedural math + Escher when the asset is a system. ZBrush when the piece is truly sculptural. Then retopo / UVs, Substance / trim, Unreal for lighting proof.

01

Blender base + sculpt

Poly-model the silhouette, then sculpt pass — ornament rhythm, edge break-up, hand form.

02

Procedural / Escher

Math-driven generation when pattern logic beats clay.

03

ZBrush · sculptural only

Reserved for true sculptural elements — not the default kitbash lane.

04

Texture + Unreal

Retopo / UVs, Painter or trim, then material instances under Melodia lighting.
