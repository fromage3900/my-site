Melodia Wix Asset Catalog

Publish this repository with GitHub Pages, then use each file URL as a Wix HTML iframe source. Query strings customize copy and data without creating duplicate files.

Hero

melodia-hero-embed.html?theme=dark&title=Melodia

Project metadata card. Params: theme, project, category, version, triangles, textures, materials, software, engine, date.
